    • rustfare: Toolbox for R language for open welfare data from Russian Federation

    rustfare,russia,russian federation,open data,R,package

    Project maintained by muuankarski Hosted on GitHub Pages — Theme by mattgraham

    The rustfare package pools together tools for searching, extracting, and formatting welfare data on Russian Federation from different data providers. The rustfare functions are named according to data provider or the data set. Currently there is data available from: Rosstat Regional Statistics, OECD, Quality of Governance -institute and World Bank
